Cleaning and Viewing a Netflix Dataset

Project Description

This project focuses on cleaning, transforming and visualizing a Netflix data set. The main goal is to prepare the data for further analysis, ensuring that it is clean, complete and in a format suitable for analysis. Additionally, some exploratory visualizations are performed to gain an initial understanding of the characteristics of the data.

Tools Used

  • Python
  • Pandas
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

Project Steps

  1. Dataset loading:
    • The netflix_titles.csv CSV file is loaded using Pandas, a data analysis library in Python.
  2. Identification and Handling of Null Values:
    • Null values ​​in the dataset are identified using the Pandas isnull() method.
    • Rows containing null values ​​are removed to ensure that only complete data is worked with.
  3. Handling Incomplete Values:
    • Null (incomplete) values ​​are counted for each column in the dataset.
    • Null values ​​in numeric columns are replaced with the average of those columns using Pandas' fillna() method.
  4. Elimination of Unnecessary Columns:
    • All columns of the dataset are reviewed to identify those that do not provide relevant information to the analysis.
    • Columns identified as unnecessary are removed.
  5. Data Type Conversion:
    • The data type of the release_year column is verified.
    • The release_year column is converted to an integer (int) type to ensure consistent formatting and facilitate subsequent analysis.
  6. Column Transformation:
    • The listed_in column contains multiple values ​​separated by commas.
    • A new column first_listed_in is created containing only the first listed_in value to simplify category analysis.

Data Exploration and Visualization

To gain an initial understanding of the data, the following visualizations were performed using Matplotlib and Seaborn:

  1. Histogram of 'release_year':
    • A histogram was generated to visualize the distribution of the release years of Netflix titles in the dataset.
    • This helps identify patterns and trends in release frequency over time.
  2. Bar chart for 'first_listed_in':
    • A bar chart was created to show the frequency of the top categories in the first_listed_in column.
    • This visualization allows you to understand what types of content are most common in the dataset.
  3. Correlation Heat Map:
    • A heat map was generated to visualize the correlations between the numerical variables of the dataset.
    • The heat map helps identify strong or weak relationships between different variables, which can be useful for more advanced analyses.

Saving the Clean Dataframe

Finally, the clean dataframe was saved to a new CSV file (Netflix_tittle_clean) for future analysis and visualization.

Conclusion

This project demonstrates essential skills in data cleansing and transformation, as well as creating exploratory visualizations. Data cleaning and preparation are critical steps in any data analysis project, as they ensure the quality and reliability of subsequent analyses. Initial visualizations provide a quick and effective understanding of the key characteristics of the dataset.
