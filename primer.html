Limpieza y Visualización de un Dataset de Netflix

Descripción del Proyecto

Este proyecto se centra en la limpieza, transformación y visualización de un conjunto de datos de Netflix. El objetivo principal es preparar los datos para un análisis más profundo, asegurando que estén limpios, completos y en un formato adecuado para su análisis. Además, se realizan algunas visualizaciones exploratorias para obtener una comprensión inicial de las características de los datos.

Herramientas Utilizadas

  • Python
  • Pandas
  • Matplotlib
  • Seaborn
  • Jupyter Notebook

Pasos del Proyecto

  1. Carga del Dataset:
    • El archivo CSV netflix_titles.csv se carga utilizando Pandas, una biblioteca de análisis de datos en Python.
  2. Identificación y Manejo de Valores Nulos:
    • Se identifican los valores nulos en el dataset utilizando el método isnull() de Pandas.
    • Se eliminan las filas que contienen valores nulos para asegurarse de que solo se trabajen con datos completos.
  3. Manejo de Valores Incompletos:
    • Se cuentan los valores nulos (incompletos) para cada columna del dataset.
    • Se reemplazan los valores nulos en las columnas numéricas con la media de dichas columnas utilizando el método fillna() de Pandas.
  4. Eliminación de Columnas Innecesarias:
    • Se revisan todas las columnas del dataset para identificar aquellas que no aportan información relevante al análisis.
    • Se eliminan las columnas identificadas como innecesarias.
  5. Conversión de Tipos de Datos:
    • Se verifica el tipo de dato de la columna release_year.
    • Se convierte la columna release_year a tipo entero (int) para asegurar un formato consistente y facilitar el análisis posterior.
  6. Transformación de Columnas:
    • La columna listed_in contiene múltiples valores separados por comas.
    • Se crea una nueva columna first_listed_in que contiene solo el primer valor de listed_in para simplificar el análisis de categorías.

Exploración de Datos y Visualización

Para obtener una comprensión inicial de los datos, se realizaron las siguientes visualizaciones utilizando Matplotlib y Seaborn:

  1. Histograma de 'release_year':
    • Se generó un histograma para visualizar la distribución de los años de lanzamiento de los títulos de Netflix en el dataset.
    • Esto ayuda a identificar patrones y tendencias en la frecuencia de lanzamientos a lo largo del tiempo.
  2. Gráfico de Barras para 'first_listed_in':
    • Se creó un gráfico de barras para mostrar la frecuencia de las categorías principales en la columna first_listed_in.
    • Esta visualización permite entender qué tipos de contenido son más comunes en el dataset.
  3. Mapa de Calor de Correlaciones:
    • Se generó un mapa de calor para visualizar las correlaciones entre las variables numéricas del dataset.
    • El mapa de calor ayuda a identificar relaciones fuertes o débiles entre diferentes variables, lo cual puede ser útil para análisis más avanzados.

Guardado del Dataframe Limpio

Finalmente, el dataframe limpio se guardó en un nuevo archivo CSV (Netflix_tittle_limpio) para futuros análisis y visualizaciones.

Conclusión

Este proyecto demuestra habilidades esenciales en la limpieza y transformación de datos, así como en la creación de visualizaciones exploratorias. La limpieza y preparación de datos son pasos críticos en cualquier proyecto de análisis de datos, ya que aseguran la calidad y fiabilidad de los análisis posteriores. Las visualizaciones iniciales proporcionan una comprensión rápida y efectiva de las características clave del dataset.
