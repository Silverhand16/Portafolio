Viewing Product Data on Amazon

Project Description

This project focuses on the collection, cleaning, analysis and visualization of product data available on Amazon. Using multiple CSV files containing various information about products from different categories and subcategories, the main objective is to extract valuable insights about product prices and ratings.

Project Objectives

  • Consolidate Data: Combine multiple CSV files into a single DataFrame for easy analysis.
  • Data Cleansing: Ensure that data is consistent and suitable for analysis, including converting prices to numerical formats.
  • Exploratory Analysis: Explore the relationships between different variables, such as price and rating, and analyze the price range by subcategory.
  • Data Visualization: Create effective visualizations to communicate findings clearly and concisely.

Tools Used

  • Python: Programming language used for data manipulation and analysis.
  • pandas: Python library for managing and analyzing data structures, particularly useful for manipulating tabular data.
  • numpy: Python library for numerical operations and array manipulation.
  • matplotlib: Python library for creating static, animated and interactive graphs.
  • seaborn: Python library based on matplotlib to improve the aesthetics and functionality of visualizations.

Project Process

  1. Data Loading and Combining:
    • Data was loaded from multiple CSV files representing different product categories on Amazon.
    • Each CSV file was read into a pandas DataFrame and stored in a dictionary for easy manipulation.
    • All information was combined into a single consolidated DataFrame.
  2. Data Cleaning:
    • Conversion of prices from string format (such as "₹32,999") to integers was carried out to facilitate numerical analysis.
    • Renamed price columns and removed rows with missing values ​​in critical columns.
    • Added a category column based on the file name to identify the category of each product.
  3. Exploratory Analysis:
    • The existence of a correlation between price and product rating was investigated using correlation coefficients.
    • The price range was analyzed based on the subcategories to identify the subcategories with the most expensive and cheapest products.
    • The average price of air conditioners was calculated and the distribution of prices was visualized.
  4. Data Visualization:
    • A stacked bar chart was created to show price ranges (minimum and maximum) by subcategory.
    • Subcategory labels have been adjusted to ensure clear and legible display.
    • A histogram was generated to visualize the price distribution of air conditioners, including a density line for a better understanding of the distribution.

Results and Findings

  • Correlation between Price and Rating: It was found that there is no significant correlation between price and rating of products, indicating that price does not necessarily influence how users rate products .
  • Price Range by Subcategory: The subcategories with the most expensive and cheapest products were identified and displayed. This can help sellers and buyers better understand the Amazon marketplace.
  • Average Price of Air Conditioners: It was calculated that the average price of air conditioners is a useful metric to compare and analyze the market for these products on Amazon.
  • Air Conditioner Price Distribution: The price distribution visualization showed how air conditioner prices are distributed, providing a clear view of trends and variations in the market. li>

Conclusion

This Amazon product data analysis project provides a detailed view of the market in terms of prices and ratings. Consolidating and cleaning data enables precise exploratory analysis, and visualizations help communicate findings effectively. These insights can be useful for both sellers and buyers on Amazon.

This project highlights key skills in data manipulation and analysis using Python, as well as creating clear, informative visualizations.
