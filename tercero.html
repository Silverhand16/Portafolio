Visualizacion de Datos de Productos en Amazon

Descripción del Proyecto

Este proyecto se centra en la recopilación, limpieza, análisis y visualización de datos de productos disponibles en Amazon. Utilizando múltiples archivos CSV que contienen información diversa sobre productos de diferentes categorías y subcategorías, el objetivo principal es extraer insights valiosos sobre los precios y las calificaciones de los productos.

Objetivos del Proyecto

  • Consolidar Datos: Combinar múltiples archivos CSV en un único DataFrame para facilitar el análisis.
  • Limpieza de Datos: Asegurar que los datos sean consistentes y adecuados para el análisis, incluyendo la conversión de precios a formatos numéricos.
  • Análisis Exploratorio: Explorar las relaciones entre diferentes variables, como precio y calificación, y analizar el rango de precios por subcategoría.
  • Visualización de Datos: Crear visualizaciones efectivas para comunicar los hallazgos de manera clara y concisa.

Herramientas Utilizadas

  • Python: Lenguaje de programación utilizado para la manipulación y análisis de datos.
  • pandas: Biblioteca de Python para el manejo y análisis de estructuras de datos, particularmente útil para la manipulación de datos tabulares.
  • numpy: Biblioteca de Python para operaciones numéricas y manipulación de arrays.
  • matplotlib: Biblioteca de Python para la creación de gráficos estáticos, animados e interactivos.
  • seaborn: Biblioteca de Python basada en matplotlib para mejorar la estética y funcionalidad de las visualizaciones.

Proceso del Proyecto

  1. Carga y Combinación de Datos:
    • Los datos se cargaron desde múltiples archivos CSV que representan diferentes categorías de productos en Amazon.
    • Cada archivo CSV se leyó en un DataFrame de pandas y se almacenó en un diccionario para una fácil manipulación.
    • Se combinó toda la información en un único DataFrame consolidado.
  2. Limpieza de Datos:
    • Se realizó la conversión de los precios de formato string (como "₹32,999") a enteros para facilitar los análisis numéricos.
    • Se renombraron las columnas de precio y se eliminaron filas con valores faltantes en las columnas críticas.
    • Se añadió una columna category basada en el nombre del archivo para identificar la categoría de cada producto.
  3. Análisis Exploratorio:
    • Se investigó la existencia de una correlación entre el precio y la calificación de los productos utilizando coeficientes de correlación.
    • Se analizó el rango de precios en función de las subcategorías para identificar las subcategorías con productos más caros y más baratos.
    • Se calculó el precio promedio de los aires acondicionados y se visualizó la distribución de los precios.
  4. Visualización de Datos:
    • Se creó un gráfico de barras apiladas para mostrar los rangos de precios (mínimo y máximo) por subcategoría.
    • Las etiquetas de las subcategorías se ajustaron para asegurar una visualización clara y legible.
    • Se generó un histograma para visualizar la distribución de precios de los aires acondicionados, incluyendo una línea de densidad para una mejor comprensión de la distribución.

Resultados y Hallazgos

  • Correlación entre Precio y Calificación: Se encontró que no existe una correlación significativa entre el precio y la calificación de los productos, lo que indica que el precio no necesariamente influye en cómo los usuarios valoran los productos.
  • Rango de Precios por Subcategoría: Las subcategorías con los productos más caros y más baratos se identificaron y visualizaron. Esto puede ayudar a los vendedores y compradores a entender mejor el mercado de Amazon.
  • Precio Promedio de Aires Acondicionados: Se calculó que el precio promedio de los aires acondicionados es una métrica útil para comparar y analizar el mercado de estos productos en Amazon.
  • Distribución de Precios de Aires Acondicionados: La visualización de la distribución de precios mostró cómo se distribuyen los precios de los aires acondicionados, proporcionando una visión clara de las tendencias y variaciones en el mercado.

Conclusión

Este proyecto de análisis de datos de productos en Amazon proporciona una visión detallada del mercado en términos de precios y calificaciones. La consolidación y limpieza de datos permiten realizar análisis exploratorios precisos, y las visualizaciones ayudan a comunicar los hallazgos de manera efectiva. Estos insights pueden ser útiles tanto para vendedores como para compradores en Amazon.

Este proyecto destaca habilidades clave en manipulación y análisis de datos utilizando Python, así como en la creación de visualizaciones informativas y claras.
